# AroundCities Project Summary (For New Chat Context)

## Project Overview

AroundCities is a local city-focused discovery platform starting with Kuching (KCH) as the first city.

Current direction:

* Focus on Kuching first
* Expand to other cities later
* Main goal is NOT just event listings
* Long-term vision is a localized social discovery/news/community content platform

The desired user experience is:

* “What’s happening around Kuching right now?”
* Similar feeling to browsing active Kuching Facebook groups/pages
* Events + local happenings + food + infrastructure + culture + community updates + lifestyle content

Examples of inspiration:

* Kuching Talk
* Sarawak Eye
* Borneo Food Hunter
* Local Kuching Facebook groups/pages

User wants AroundCities to eventually become a centralized local-content hub without the noise of worldwide Facebook content.

# Important Database Issue Encountered

Duplicate foreign keys were accidentally created after using another ChatGPT conversation.

This caused Supabase embedded queries to fail with:

```text
Could not embed because more than one relationship was found
```

Fix:
All embedded relationships must explicitly specify FK names.

Correct query pattern:

```ts
.select(`
  *,
  event_translations!event_translations_event_id_fkey(*),
  event_sessions!event_sessions_event_id_fkey(*),
  event_sources!event_sources_event_id_fkey(*)
`)
```

This is VERY IMPORTANT for future queries.

# Next.js Version Issue Encountered

Using latest Next.js App Router.

Problem:
`params.id` caused errors because params became async.

Fix pattern:

```ts
export default async function Page({
  params,
}: {
  params: Promise<{ id: string }>;
}) {
  const { id } = await params;
}
```

Important for all `[id]` routes.
